# server-github-alumne03
Projecto de servidor linux al nuvol amb GitHub Codespaces, SSH i Apache
Este projecto consiste en la creacion i configuracion de un servidor linux al nuvol utilizando GitHub Codespaces.
Inclou:

. Acesso remoto con SSH
. Instalacion i configuracion de apache
. web publica
. web privada con autenticacion basica

El repositorio contiene la estructura basica del projecto i el fitxero necesario para el desplegamiento.
